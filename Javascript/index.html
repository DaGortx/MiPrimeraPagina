El DOM (Document Object Model, Modelo de Objetos del Documento) es una interfaz de programación que representa el contenido de una página web (HTML o XML) como una estructura de árbol de objetos. Funciona como un puente entre el código de un documento y el script que se está ejecutando, permitiendo que el código manipule el contenido, la estructura y el estilo de la página en tiempo real, como por ejemplo, para modificar elementos, añadir o eliminar contenido, y responder a eventos del usuario.

JavaScript (JS) es un lenguaje de programación de alto nivel que se utiliza principalmente para hacer las páginas web interactivas y dinámicas. Permite crear funciones como actualizar contenido de redes sociales, mostrar animaciones, validar formularios y responder a la interacción del usuario, como clics y movimientos del ratón. A pesar de llamarse así, no debe confundirse con el lenguaje Java, y se usa tanto en el navegador (frontend) como en servidores (backend).
